how to run:
1. start mongodb service
2. npm install
3. npm start
4. open http://localhost:3000/

folders/files desc:
1. server/index.js: 服务端入口
2. server/controllers/*.js: 路由控制器（bug/user）
3. server/models/*.js: Mongoose Model（bug/user）
4. src/index.js: 客户端入口
5. src/App.js: 客户端路由主文件
6. src/Pages/*.js: 客户端路由页面文件（主页、增加用户、增加或编辑bug、查看bug详情）
7. src/utils.js: 工具方法
8. src/index.css: 样式文件
